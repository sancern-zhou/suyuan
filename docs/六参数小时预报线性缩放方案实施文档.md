六参数小时预报线性缩放方案实施文档

  1 概述

  1.1 背景

  当前系统已实现基于Open-Meteo的小时AQI预报，并使用SQL
  Server日AQI预报数据进行校准。为满足六项污染物指标（PM2.5、PM10、O3、SO2、NO2、CO）同步预报的需求，需扩展系统能力，实现六参数的线性缩放校准，并与现有日预报数据保持一致。

  1.2 目标

  - 实现六参数小时预报数据（未来5天）的线性缩放校准
  - 确保校准后的六参数日均值计算出的AQI落在SQL Server日预报AQI范围内
  - 保持六参数的日内波动特征和相互比例关系
  - 复用现有小时预报表结构进行数据存储

  1.3 范围

  本文档定义了六参数线性缩放校准的技术方案，包括算法设计、数据流程、数据库变更和API扩展。

  3 详细设计

  3.1 AQI ↔ PM2.5 转换标准

  基于中国GB 3095-2012《环境空气质量标准》，AQI与PM2.5浓度对应关系如下：

  | AQI范围   | 等级   | PM2.5日均浓度 (μg/m³) | PM2.5小时浓度 (μg/m³) |
  |---------|------|-------------------|-------------------|
  | 0-50    | 优    | 0-35              | 0-50              |
  | 51-100  | 良    | 35-75             | 50-100            |
  | 101-150 | 轻度污染 | 75-115            | 100-150           |
  | 151-200 | 中度污染 | 115-150           | 150-250           |
  | 201-300 | 重度污染 | 150-250           | 250-350           |
  | 301-500 | 严重污染 | 250-500           | 350-500           |

  线性插值公式：

  PM2.5 = PM_low + (AQI - AQI_low) × (PM_high - PM_low) / (AQI_high - AQI_low)
  AQI = AQI_low + (PM2.5 - PM_low) × (AQI_high - AQI_low) / (PM_high - PM_low)

  3.2 线性缩放算法

  3.2.1 算法描述

  1. 获取目标PM2.5日均浓度范围
    - 输入：SQL Server日预报 MinAqi, MaxAqi
    - 转换：使用线性插值计算对应的PM2.5浓度范围
  2. 计算Open-Meteo六参数日均值
    - 按日期分组计算各参数的平均值
    - 六参数独立计算
  3. 计算缩放因子
    - 计算目标中心值：targetCenter = (targetPm25Min + targetPm25Max) / 2
    - 计算当前PM2.5日均值：currentMean
    - 计算缩放因子：scaleFactor = targetCenter / currentMean
    - 限制缩放因子范围：[0.5, 2.0]
  4. 应用缩放因子
    - 六参数采用相同缩放因子
    - 保持参数间的相对比例关系
  5. 边界约束
    - 各参数不超过中国二级标准的1.5倍
    - PM2.5 ≤ 115 μg/m³, PM10 ≤ 250 μg/m³ 等

  3.2.2 算法伪代码

  FUNCTION calibrateSixParams(hourlyData, sqlServerDaily):
      // 输入验证
      IF sqlServerDaily.MinAqi IS NULL OR sqlServerDaily.MaxAqi IS NULL:
          RETURN hourlyData  // 无需校准

      // Step 1: AQI → PM2.5 浓度范围
      targetPm25Range = aqiToPm25Range(sqlServerDaily.MinAqi, sqlServerDaily.MaxAqi)
      targetCenter = (targetPm25Range.min + targetPm25Range.max) / 2

      // Step 2: 按日期分组计算六参数日均值
      dailyMeans = calculateDailyMeans(hourlyData)

      // Step 3: 对每一天计算缩放因子
      calibratedData = []

      FOR EACH date IN dailyMeans:
          currentPm25 = dailyMeans[date].pm25

          // 计算缩放因子
          scaleFactor = targetCenter / currentPm25

          // 限制缩放因子范围
          scaleFactor = clamp(scaleFactor, 0.5, 2.0)

          // Step 4: 应用缩放
          FOR EACH hour IN hourlyData WHERE hour.date == date:
              calibratedHour = {
                  time: hour.time,
                  pm25: hour.pm25 * scaleFactor,
                  pm10: hour.pm10 * scaleFactor,
                  o3: hour.o3 * scaleFactor,
                  so2: hour.so2 * scaleFactor,
                  no2: hour.no2 * scaleFactor,
                  co: hour.co * scaleFactor,
                  scale_factor: scaleFactor
              }

              // Step 5: 边界约束
              calibratedHour = applyBoundaryConstraints(calibratedHour)

              calibratedData.append(calibratedHour)
          END FOR
      END FOR

      RETURN calibratedData
  END FUNCTION

  async saveHourly(hourlyData, scheme = 'default') {

          ? 'air_quality_hourly'
          : `air_quality_hourly_${scheme}`;

      const values = hourlyData.map(h => `
          (
              '${h.forecast_time}',
              '${h.source}',
              ${h.aqi},
              ${h.level?.value || 'NULL'},
              '${h.level?.name || ''}',
              '${h.level?.class || ''}',
              '${h.primary_pollutant || ''}',
              ${h.raw_aqi || 'NULL'},
              ${h.shift_value || 'NULL'},
              ${h.scale_factor || 'NULL'},
              ${h.pm25 || 'NULL'},
              ${h.pm10 || 'NULL'},
              ${h.o3 || 'NULL'},
              ${h.so2 || 'NULL'},
              ${h.no2 || 'NULL'},
              ${h.co || 'NULL'}
          )
      `).join(',');

      const query = `
          INSERT INTO ${tableName} (
              forecast_time, source, aqi, level_value, level_name,
              level_class, primary_pollutant, raw_aqi, shift_value,
              scale_factor, pm25, pm10, o3, so2, no2, co
          ) VALUES ${values}
          ON CONFLICT (forecast_time) DO UPDATE SET
              aqi = EXCLUDED.aqi,
              pm25 = EXCLUDED.pm25,
              pm10 = EXCLUDED.pm10,
              o3 = EXCLUDED.o3,
              so2 = EXCLUDED.so2,
              no2 = EXCLUDED.no2,
              co = EXCLUDED.co,
              updated_at = CURRENT_TIMESTAMP
      `;

      return this.pool.query(query);

  8.2 AQI 计算参数权重

  | 污染物   | IAQI 公式系数                           |
  |-------|-------------------------------------|
  | PM2.5 | PM2.5 = (PM2.5,24h - 35) × 100 / 40 |
  | PM10  | PM10 = (PM10,24h - 50) × 100 / 50   |
  | O3    | O3 = (O3,8h - 100) × 100 / 60       |
  | SO2   | SO2 = (SO2,24h - 50) × 100 / 10     |
  | NO2   | NO2 = (NO2,24h - 40) × 100 / 40     |
  | CO    | CO = (CO,24h - 2) × 100 / 2         |

  AQI = max(PM2.5, PM10, O3, SO2, NO2, CO)
