# 气象专家模式实现文档

## 概述

气象专家模式是ReAct Agent系统中的单专家模式，专注于气象条件分析和可视化展示。该模式遵循"专注气象 + 默认可视化"的设计原则，为用户提供专业的 meteorological analysis services。

## 核心特性

### 1. 专注气象分析
- **纯气象数据**: 仅处理气象相关工具，不包含空气质量数据（由组分专家负责）
- **专业工具集**: 集成ERA5历史气象数据、轨迹分析、上风向企业分析等
- **智能场景选择**: 支持4种分析模式（基础、增强、轨迹、综合）

### 2. 默认可视化集成
- **自动图表生成**: 默认包含 `smart_chart_generator` 工具
- **多图表类型**: 支持时序图、风向玫瑰图、轨迹地图等15种图表
- **专业可视化**: 气象轨迹地图、传输路径图等专用可视化

### 3. 多层级工具架构
```
基础层 (Core Layer)
├── get_weather_data           # ERA5历史气象数据
├── get_universal_meteorology  # 通用气象接口
├── get_current_weather        # 当前天气数据（Open-Meteo实时）
├── get_weather_forecast       # 天气预报（Open-Meteo预报）
└── get_tianji_weather         # 中科天机高精度预报（19压力层、45天预报）

增强层 (Enhancement Layer)
├── get_fire_hotspots          # NASA FIRMS火点数据
├── get_dust_data              # CAMS沙尘预报数据
└── get_satellite_data         # 卫星遥感数据

高级层 (Advanced Layer)
├── meteorological_trajectory_analysis  # 后向轨迹分析（集成Tianji/GDAS/GFS）
├── trajectory_simulation              # 轨迹模拟
└── analyze_upwind_enterprises         # 上风向企业分析

可视化层 (Visualization Layer)
├── smart_chart_generator    # 智能图表生成（默认可视化）
├── generate_chart           # 自定义图表
└── generate_map            # 轨迹地图专用
```

**底层数据客户端**：
- `TianjiWeatherClient` - 中科天机API客户端（19压力层、45天预报、15分钟分辨率）
- `GDASGFSClient` - NOAA GDAS/GFS数据下载器（HYSPLIT轨迹分析专用）
- `OpenMeteoClient` - Open-Meteo API客户端（实时天气、预报、ERA5历史）

数据质量等级

- A级: ERA5(再分析数据，权威)、WeatherAPI(真实观测)、GDAS(NOAA官方分析)
- B级: Open-Meteo(模型数据，精度良好)、Tianji(高分辨率预报)、GFS(全球预报标准)
- C级: CAMS沙尘(预报数据，参考级别)

注意：
1. 边界层高度是污染分析的关键参数，仅在Open-Meteo预报数据中提供，对逆温、污染扩散判断至关重要
2. 轨迹分析优先使用Tianji（预报场景）或GDAS/ERA5（历史场景），它们提供完整的19压力层高空风数据
3. Tianji支持最长45天预报，适合长期污染趋势分析
