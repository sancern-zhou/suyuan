## 🔍 如何查看图表

### 方法1：发送会生成图表的查询
在输入框中输入类似以下查询：
```
"绘制一个柱状图"
"分析数据并生成图表"
"显示PM2.5的时间趋势图"
"生成饼图展示污染物占比"
```

### 方法2：浏览器控制台调试
1. 打开浏览器开发者工具 (F12)
2. 切换到 Console 标签
3. 执行以下命令查看状态：
```javascript
// 查看当前可视化数据
console.log('currentVisualization:', store.currentVisualization)

// 查看所有图表
console.log('charts:', store.results.charts)

// 查看结果总数
console.log('results:', store.results)
```

### 方法3：手动设置测试图表
在浏览器控制台中执行：
```javascript
// 设置一个测试图表
store.currentVisualization = {
  type: 'chart',
  data: {
    type: 'bar',
    x: ['周一', '周二', '周三', '周四', '周五'],
    y: [12, 19, 23, 25, 22]
  },
  title: '测试柱状图'
}
```

## 🐛 常见问题与解决

### 问题1：看不到可视化面板
**现象**：界面右侧没有显示任何面板

**原因**：
- `currentVisualization` 为 null
- 还没有分析结果

**解决**：
```javascript
// 检查状态
console.log('currentVisualization:', store.currentVisualization)
console.log('hasResults:', store.hasResults)

// 如果是null，手动设置一个测试图表
store.currentVisualization = { type: 'chart', data: {...} }
```

### 问题2：面板显示"暂无可视化内容"
**现象**：面板存在但显示空状态

**原因**：
- 后端没有返回图表数据
- 数据格式不正确
- 没有触发complete事件

**解决**：
1. 查看控制台日志
2. 检查后端是否返回visualization数据
3. 启用调试模式查看详细日志

### 问题3：面板显示但图表是空白
**现象**：面板有内容但图表区域空白

**原因**：
- ChartPanel接收的数据格式不正确
- ECharts初始化失败
- 图表配置错误

**解决**：
1. 查看调试信息（面板中的"调试信息"折叠面板）
2. 检查数据格式
3. 查看浏览器控制台错误

## 🔧 修复记录

### 已修复的问题

#### 1. 数据格式问题
**原问题**：图表数据没有正确的`type`字段

**修复**：在`handleResult`中强制设置正确的结构
```javascript
this.currentVisualization = {
  type: 'chart',              // ✅ 添加type字段
  data: chartData.data || chartData.payload || chartData,
  title: chartData.title || '图表',
  meta: chartData.meta || {}
}
```

#### 2. 事件覆盖问题
**原问题**：`complete`事件中重复设置`currentVisualization`

**修复**：移除重复设置，让`handleResult`统一处理
```javascript
// 修复前
this.currentVisualization = data.visualization  // ❌ 提前设置
this.handleResult(data.visualization)

// 修复后
this.handleResult(data.visualization)  // ✅ 统一处理
```

#### 3. 面板显示条件
**原问题**：`v-if="store.currentVisualization"`导致面板不显示

**修复**：移除条件，让面板始终显示便于调试
```vue
<!-- 修复前 -->
<VisualizationPanel v-if="store.currentVisualization" />

<!-- 修复后 -->
<VisualizationPanel />
```

## 📊 数据流

### 图表数据流向
```
后端API
  ↓ 返回SSE事件
handleEvent('result' | 'complete')
  ↓
handleResult(data)
  ↓ 设置
currentVisualization = {
  type: 'chart',
  data: {...},
  title: '...'
}
  ↓
VisualizationPanel
  ↓ 条件判断
content.type === 'chart'
  ↓
ChartPanel
  ↓
ECharts渲染
```

### 关键日志位置
在浏览器控制台中查找：
- `[handleResult] 处理结果:` - 接收到的数据
- `[handleResult] 设置图表可视化:` - 设置的currentVisualization
- `[ChartPanel]` - 图表组件日志
