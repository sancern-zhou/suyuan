# 大气污染溯源分析系统 - 前端重构

## 项目简介

本项目是针对大气污染溯源分析系统的前端重构，采用极简对话界面设计，参考MANUS的简洁风格，专注于提供清晰、高效的用户体验。

## 核心特性

- ✅ **极简对话流** - ChatGPT风格，纯文字展示，去除所有装饰
- ✅ **随时打断** - 分析过程中可随时输入新问题
- ✅ **动态可视化** - 图表/地图按需显示，右侧面板动态展示
- ✅ **调试功能** - 可选显示LLM上下文和工具调用信息
- ✅ **成果报告** - 独立HTML报告，新窗口打开
- ✅ **响应式设计** - 适配桌面端、平板端、手机端

## 技术栈

- **框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **状态管理**: Pinia
- **路由**: Vue Router 4
- **图表库**: ECharts 5
- **样式**: SCSS

## 快速开始

### 安装依赖

```bash
cd frontend
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 `http://localhost:5174`

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash

```

## 功能说明

### 1. 极简对话界面

- **无图标设计**: 去除所有Emoji和装饰图标
- **纯文字对话**: 清晰展示用户与Agent的对话
- **时间戳**: 每条消息带有精确到秒的时间戳
- **消息链接**: 每条消息可生成分享链接

### 2. 随时打断功能

- **输入框常驻**: 始终可用，不在分析时禁用
- **新问题提示**: 分析进行时输入新问题会提示确认
- **快捷键支持**: Enter发送，Shift+Enter换行

### 3. 动态可视化面板

- **按需显示**: 只在有可视化内容时显示
- **多类型支持**: 图表、地图、数据表、图片
- **空状态提示**: 无内容时显示友好提示

### 4. 调试功能

- **调试开关**: 顶部按钮可开启/关闭调试模式
- **内嵌显示**: 调试信息以可折叠形式嵌入消息
- **详细信息**: 显示LLM上下文、工具调用、API响应

### 5. 成果报告

- **独立报告**: 点击按钮生成独立HTML报告
- **新窗口打开**: 不影响当前对话
- **专业格式**: 包含执行摘要、关键指标、详细分析

## 设计理念

### 极简主义

去除一切不必要的装饰元素，专注于内容本身。使用简洁的配色方案、清晰的字体层次、规范的间距系统。

### 用户中心

以用户需求为中心，提供最直观、最易用的交互方式。随时打断、智能溯源、动态展示等特性均以提升用户体验为目标。

### 专业呈现

去娱乐化设计，使用专业图标和配色，符合企业级应用标准。避免过度动画和特效，保持界面稳定、专业。

## 响应式设计

- **桌面端 (>1024px)**: 左侧对话流 + 右侧可视化面板
- **平板端 (768-1024px)**: 可折叠右侧面板
- **手机端 (<768px)**: 全屏对话，可视化面板以弹窗形式出现

## 开发规范

### 组件命名

- 使用PascalCase命名组件
- 组件名使用完整单词，避免缩写
- 遵循Vue官方风格指南

### 样式规范

- 使用SCSS预处理器
- 遵循BEM命名规范
- 使用flexbox布局
- 颜色使用设计系统定义

### 状态管理

- 使用Pinia进行状态管理
- 状态结构扁平化，避免深层嵌套
- 合理使用getters和actions
