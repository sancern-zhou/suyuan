## 📝 使用方法

### 启用6色阶模式

```python
from app.tools.visualization.polar_contour_generator import generate_pollution_rose_contour
import tempfile
import base64
import json

# 加载数据
data_file = '/home/xckj/suyuan/backend_data_registry/data_registry/{data_id}.json'
with open(data_file, 'r') as f:
    data = json.load(f)

# 提取字段
wind_dirs = [record['wind_direction_10m'] for record in data]
wind_speeds = [record['wind_speed_10m'] for record in data]
concentrations = [record['PM2_5'] for record in data]

# 生成6色阶模式的极坐标图
img_base64 = generate_pollution_rose_contour(
    wind_directions=wind_dirs,
    wind_speeds=wind_speeds,
    concentrations=concentrations,
    title="PM2.5浓度6色阶污染玫瑰图（广雅中学）",
    pollutant_name="PM2.5",
    unit="μg/m³",
    use_six_level=True,       # ← 启用6色阶模式
    value_range=(0, 250),    # ← PM2.5标准范围
    grid_resolution=100,
    interpolation_method="cubic"
)

# 保存并触发缓存
with tempfile.NamedTemporaryFile(suffix=".png", delete=False) as f:
    f.write(base64.b64decode(img_base64))
    print(f"CHART_SAVED:{f.name}")
```

### 默认模式 vs 6色阶模式

| 特性 | 默认模式 | 6色阶模式 |
|------|----------|------------|
| **色阶数量** | 20层连续渐变 | 6个固定等级 |
| **颜色方案** | RdYlBu_r（深蓝→深红） | 绿黄橙红紫褐红 |
| **颜色条** | 数值刻度 | 等级标签（优/良/轻度/中度/重度/严重） |
| **标准化** | 相对色阶（根据数据范围） | 绝对标尺（基于国家标准） |
| **适用场景** | 数据探索、相对比较 | 报告生成、合规性评估 |

## 📊 数据流程

```
原始数据（可能包含负值和异常值）
    ↓
【阶段1】基础验证（过滤NaN、风速≤0）
    ↓
【阶段2】智能过滤
  - 负值 → NaN
  - IQR异常值检测（Q3 + 3*IQR）
  - 过滤率 ~99%
    ↓
【阶段3】确定色阶范围
  - 6色阶：使用污染物标准限值
  - 默认：5-95分位数或用户指定
    ↓
【阶段4】插值计算
  - scipy.interpolate.griddata (cubic)
  - 可能产生过冲值
    ↓
【阶段5】裁剪到[vmin, vmax]
  - np.clip确保不超出色阶范围
    ↓
【阶段6】绘制图表
  - 6色阶：BoundaryNorm + 自定义colormap
  - 默认：连续colormap (RdYlBu_r)
    ↓
输出图表
```
