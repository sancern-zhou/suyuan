按照用户要求，已参考VOCs和颗粒物工具的实现方式，将工具改为**直接接受自然语言输入**的查询方式。

## 📝 实现对比

### 旧版本（结构化参数）❌
```python

    include_coordinates=True

# 工具内部转换为自然语言
```

### 新版本（自然语言输入）✅
```python

    question="查询广州的国控监测站点"
)

response = await http_client.post(url, json_data={
```

## 🔧 核心技术改进

### 1. 参数变更
| 旧版本 | 新版本 |
|--------|--------|
| `city_name` (必需) | `question` (必需) |
| `station_type` (可选) | - |
| `include_coordinates` (可选) | - |

### 2. HTTP客户端
| 旧版本 | 新版本 |
|--------|--------|
| `aiohttp.ClientSession()` | `http_client.post()` |
| 自定义错误处理 | 统一的HTTP客户端重试机制 |
| 手动JSON解析 | 自动JSON解析 |

### 3. 数据提取
| 旧版本 | 新版本 |
|--------|--------|
| 自定义`_parse_uqp_response()` | 参考VOCs工具的`_seek_records()` |
| 转换为UDF格式 | 直接返回数据列表 |
| 复杂的数据转换逻辑 | 简单的数据提取逻辑 |

### 4. 返回格式
| 旧版本 | 新版本 |
|--------|--------|
| UDF v1.0统一格式 | VOCs工具风格格式 |
| `status`, `success`, `data` | `success`, `data`, `count` |
| `metadata`, `summary` | `question`, `summary` |

## 📚 参考实现

### VOCs工具参考
```python
# 参考VOCs工具的查询方式
async def _query_vocs(self, question: str) -> List[Dict[str, Any]]:
    url = f"{settings.vocs_data_api_url}/api/uqp/query"
    response = await http_client.post(url, json_data={"question": question})
    return self._extract_list(response)

# 参考VOCs工具的数据提取
def _seek_records(self, payload: Any) -> Optional[List[Dict[str, Any]]]:
    if isinstance(payload, dict):
        for key in ("data", "result", "results", "stations", "value"):
            value = payload.get(key)
            if isinstance(value, list):
                return self._ensure_dict_list(value)
```

### 我们的实现
```python
# 我们的自然语言查询实现
async def execute(self, question: str, **kwargs) -> Dict[str, Any]:
    url = "http://180.184.91.74:9091/api/uqp/query"
    request_data = {
        "question": question,
        "intent_hint": "QUERY_STATION_INFO"
    }
    response = await http_client.post(url, json_data=request_data)
    records = self._seek_records(response)
    return {
        "success": True,
        "data": records,
        "count": len(records),
        "question": question,
        "summary": f"找到 {len(records)} 个监测站点",
    }
```
