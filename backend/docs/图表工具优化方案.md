图表生成工具优化方案

## 问题分析

### 当前实现的问题

**硬编码场景**（存在于 `app/utils/visualization.py`）：
1. `generate_vocs_analysis_visuals()` - 固定生成3个图表
   - VOCs浓度饼图（Top 10）
   - OFP贡献柱状图（Top 10）
   - 行业来源饼图（Top 8）

2. `generate_particulate_analysis_visuals()` - 固定生成2个图表
   - 颗粒物组分饼图（Top 10）
   - 行业PM排放柱状图（Top 8）

3. `generate_multi_indicator_timeseries()` - 固定双轴图
   - 左轴：污染物浓度 + AQI
   - 右轴：温度、湿度、风速

4. `generate_regional_comparison_visual()` - 固定多线图
   - 目标站点 + 周边站点浓度对比

**硬编码的优势**：
- ✅ 针对特定场景深度优化（标签位置、颜色、格式）
- ✅ 配置稳定可靠
- ✅ 性能好（无需LLM决策）

**硬编码的劣势**：
- ❌ 无法适应新场景（如：比较两个时间段、显示风向玫瑰图）
- ❌ Agent无法根据数据特征动态选择图表类型
- ❌ 无法处理非标准数据结构

---

## 优化方案：两层架构

### 方案概述

采用**模板库 + 智能生成器**的两层架构：

```
图表生成工具
├── 第一层：模板库（Template Library）
│   ├── 预定义的优化模板（针对已知场景）
│   ├── 快速、可靠、深度优化
│   └── 适用于90%的常见场景
│
└── 第二层：智能生成器（Smart Generator）
    ├── LLM驱动的动态图表生成
    ├── 分析数据特征，自动选择图表类型
    └── 适用于10%的新场景或特殊需求
```

## 实现设计

### 1. 模板库（Template Library）

**目标**：保留现有硬编码优化，同时使其可扩展

**实现**：

```python
# app/tools/visualization/chart_templates.py

from enum import Enum
from typing import Dict, Any, List, Optional, Callable


class ChartTemplate(Enum):
    \"\"\"预定义的图表模板\"\"\"
    VOCS_CONCENTRATION_PIE = "vocs_concentration_pie"
    VOCS_OFP_BAR = "vocs_ofp_bar"
    VOCS_INDUSTRY_PIE = "vocs_industry_pie"
    PM_COMPONENT_PIE = "pm_component_pie"
    PM_INDUSTRY_BAR = "pm_industry_bar"
    MULTI_INDICATOR_DUAL_AXIS = "multi_indicator_dual_axis"
    REGIONAL_COMPARISON_TIMESERIES = "regional_comparison_timeseries"
    WIND_ROSE = "wind_rose"  # 新增：风向玫瑰图
    HEATMAP_TEMPORAL = "heatmap_temporal"  # 新增：时间热力图


class ChartTemplateRegistry:
    \"\"\"图表模板注册表\"\"\"

    def __init__(self):
        self._templates: Dict[str, Callable] = {}
        self._register_builtin_templates()

    def register(self, template_id: str, generator_func: Callable):
        \"\"\"注册模板生成函数\"\"\"
        self._templates[template_id] = generator_func

    def generate(
        self,
        template_id: str,
        data: Any,
        **kwargs
    ) -> Dict[str, Any]:
        \"\"\"使用模板生成图表\"\"\"
        if template_id not in self._templates:
            raise ValueError(f"未知模板: {template_id}")

        generator = self._templates[template_id]
        return generator(data, **kwargs)

    def _register_builtin_templates(self):
        \"\"\"注册内置模板（保留现有优化）\"\"\"
        # 导入现有函数
        from app.utils.visualization import (
            generate_vocs_analysis_visuals,
            generate_particulate_analysis_visuals,
            # ... 其他函数
        )

        # 注册为模板
        self.register("vocs_analysis", generate_vocs_analysis_visuals)
        self.register("pm_analysis", generate_particulate_analysis_visuals)
        # ... 注册其他模板
```

**优势**：
- ✅ 保留现有优化配置
- ✅ 可扩展（随时添加新模板）
- ✅ 模板化管理

### 2. 智能生成器（Smart Generator）

**目标**：根据数据特征和用户意图，动态生成图表

**实现**：

```python
# app/tools/visualization/generate_chart/tool.py

class GenerateChartTool(LLMTool):
    \"\"\"
    智能图表生成工具（给ReAct Agent使用）

    功能：
    1. 优先尝试使用模板库（快速、可靠）
    2. 如果没有匹配模板，使用LLM分析数据生成图表
    \"\"\"

    def __init__(self):
        function_schema = {
            "name": "generate_chart",
            "description": \"\"\"
                根据数据和需求生成ECharts图表配置。

                支持的场景：
                - VOCs分析：自动生成VOCs浓度饼图、OFP贡献柱状图
                - 颗粒物分析：自动生成颗粒物组分饼图、行业排放柱状图
                - 时序对比：生成多站点或多指标时序图
                - 自定义：根据数据特征自动选择合适的图表类型
            \"\"\",
            "parameters": {
                "type": "object",
                "properties": {
                    "data": {
                        "type": "object",
                        "description": "要可视化的数据（JSON格式）"
                    },
                    "chart_type_hint": {
                        "type": "string",
                        "description": "图表类型提示（可选）：pie, bar, line, timeseries, heatmap, scatter, radar",
                        "enum": ["pie", "bar", "line", "timeseries", "heatmap", "scatter", "radar", "auto"]
                    },
                    "scenario": {
                        "type": "string",
                        "description": "场景标识（可选）：vocs_analysis, pm_analysis, weather_analysis, regional_comparison",
                        "enum": ["vocs_analysis", "pm_analysis", "weather_analysis", "regional_comparison", "custom"]
                    },
                    "title": {
                        "type": "string",
                        "description": "图表标题（可选）"
                    },
                    "x_field": {
                        "type": "string",
                        "description": "X轴字段名（可选，用于自定义图表）"
                    },
                    "y_field": {
                        "type": "string",
                        "description": "Y轴字段名（可选，用于自定义图表）"
                    }
                },
                "required": ["data"]
            }
        }

        super().__init__(
            name="generate_chart",
            description="Generate ECharts configuration dynamically",
            category=ToolCategory.VISUALIZATION,
            function_schema=function_schema,
            version="2.0.0"
        )

        self.template_registry = ChartTemplateRegistry()

    async def execute(
        self,
        data: Dict[str, Any],
        chart_type_hint: str = "auto",
        scenario: str = "custom",
        title: Optional[str] = None,
        x_field: Optional[str] = None,
        y_field: Optional[str] = None,
        **kwargs
    ) -> Dict[str, Any]:
        \"\"\"
        执行图表生成

        工作流：
        1. 尝试匹配预定义模板（优先）
        2. 如果没有匹配，使用LLM分析数据生成图表
        \"\"\"
        try:
            # Step 1: 尝试使用模板库
            if scenario != "custom":
                try:
                    chart_config = self.template_registry.generate(
                        template_id=scenario,
                        data=data,
                        title=title
                    )

                    return {
                        "success": True,
                        "chart_config": chart_config,
                        "method": "template",
                        "template_used": scenario
                    }
                except ValueError:
                    # 模板不存在，继续到智能生成
                    pass

            # Step 2: 智能生成（LLM驱动）
            chart_config = await self._generate_with_llm(
                data=data,
                chart_type_hint=chart_type_hint,
                title=title,
                x_field=x_field,
                y_field=y_field
            )

            return {
                "success": True,
                "chart_config": chart_config,
                "method": "llm_generated",
                "chart_type": chart_type_hint
            }

        except Exception as e:
            return {
                "success": False,
                "error": str(e)
            }

    async def _generate_with_llm(
        self,
        data: Dict[str, Any],
        chart_type_hint: str,
        title: Optional[str],
        x_field: Optional[str],
        y_field: Optional[str]
    ) -> Dict[str, Any]:
        \"\"\"
        使用LLM分析数据，生成图表配置

        工作流：
        1. 分析数据结构和特征
        2. 选择最合适的图表类型
        3. 生成ECharts配置
        \"\"\"
        from app.services.llm_service import llm_service

        # 构建提示词
        prompt = f\"\"\"
你是一个数据可视化专家。请分析以下数据，生成ECharts图表配置。

# 数据
```json
{json.dumps(data, ensure_ascii=False, indent=2)[:1000]}
```

# 需求
- 图表类型提示: {chart_type_hint}
- 标题: {title or '自动生成'}
- X轴字段: {x_field or '自动识别'}
- Y轴字段: {y_field or '自动识别'}

# 任务
1. 分析数据结构，识别关键字段
2. 根据数据特征选择最合适的图表类型（饼图、柱状图、折线图、散点图等）
3. 生成完整的ECharts配置JSON

# 输出格式
返回一个JSON对象，格式如下：
{{
    "id": "chart_id",
    "type": "pie|bar|line|scatter|heatmap",
    "title": "图表标题",
    "mode": "dynamic",
    "payload": {{
        // ECharts配置
        "x": [...],  // X轴数据（可选）
        "y": [...],  // Y轴数据（可选）
        "series": [...]  // 系列数据
    }}
}}
\"\"\"

        # 调用LLM
        response = await llm_service.call_llm_with_json_response(prompt)

        return response
```

**优势**：
- ✅ 自动分析数据特征
- ✅ 智能选择图表类型
- ✅ 处理任意数据结构
