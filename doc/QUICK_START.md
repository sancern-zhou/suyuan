# 🚀 快速启动指南

## 立即开始使用实时流式分析

### 1️⃣ 启动后端（终端1）

```bash
cd backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

等待看到：
```
INFO:     Application startup complete.
INFO:     Uvicorn running on http://0.0.0.0:8000
```

### 2️⃣ 启动前端（终端2）

```bash
cd frontend
npm run dev
```

等待看到：
```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
```

### 3️⃣ 打开浏览器

访问：**http://localhost:5173**

### 4️⃣ 开始对话

在对话框中输入：

```
分析2025年8月9日广州从化天湖站点臭氧超标原因
```

或者：

```
2025年8月9日广州从化天湖站O3污染溯源分析
```

## ✨ 核心特性

### 📊 实时进度展示
- 每个API调用都有实时反馈
- LLM提取结果立即可见
- 错误信息即时显示

### 🔍 详细调试信息
- 参数提取结果（站点、污染物、时间）
- 站点信息（名称、位置、坐标）
- 数据统计（监测点数、气象点数、周边站点数）
- 企业分析（企业数量、地图状态）
- 模块生成状态（内容长度、可视化数量、置信度）

### ⚠️ 智能错误处理
- 参数提取失败 → 提示明确指定站点
- 站点未找到 → 提示检查站点名称
- 数据为空 → 提示可能原因
- API失败 → 显示具体错误信息

## 🐛 常见问题

### 前端无响应？
1. 检查后端是否运行：`python check_backend.py`
2. 查看浏览器Console（F12）
3. 检查Network标签中的请求

### 参数提取失败？
使用更明确的查询：
- ✅ "分析2025年8月9日广州从化天湖站点的O3污染"
- ❌ "分析污染"

### 站点未找到？
- 检查站点名称拼写
- 参考 `backend/test_api.py` 中的示例

### 数据为空？
- 更换时间范围
- 检查外部API是否可访问
- 查看后端日志
