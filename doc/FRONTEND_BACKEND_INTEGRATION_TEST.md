## 🐛 常见问题排查

### 问题1：前端显示"配置获取失败"
**原因**: 后端服务未启动或端口不正确

**解决**:
```bash
# 检查后端是否运行
curl http://localhost:8000/health

# 如果没有运行，启动后端
cd backend
python -m uvicorn app.main:app --reload
```

### 问题2：分析请求失败
**原因**: API路径不正确或后端错误

**解决**:
1. 打开浏览器开发者工具（F12）
2. 查看 Network 标签
3. 检查 `/api/analyze` 请求的状态码和响应
4. 查看后端日志

### 问题3：地图不显示
**原因**: 高德地图API key未配置或无效

**解决**:
1. 检查 `backend/config/settings.py` 中的 `AMAP_PUBLIC_KEY`
2. 确保API key有效且有足够的配额

### 问题4：区域对比分析没有显示
**原因**: 可能没有找到附近站点数据

**解决**:
1. 查看后端日志，搜索 `nearby_stations_with_data`
2. 确认是否找到了附近站点
3. 如果没有找到，可能是监测数据API返回为空
