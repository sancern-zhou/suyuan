## 四、图表解析结果提取流程

### 1. 提取图表引用（`_extract_chart_references`）

**位置**：`backend/app/services/report_exporter.py` 第553-602行

**方法**：
```python
def _extract_chart_references(self, section_content: str) -> List[Dict]:
    """从章节内容中提取图表引用和解析文本"""
    references = []
    
    # 方法1：从预设标识中提取 [CHART_X_START]... [CHART_X_END]
    chart_pattern = r'\[CHART_(\d+)_START\](.*?)\[CHART_\d+_END\]'
    chart_matches = re.findall(chart_pattern, section_content, re.DOTALL)
    
    for chart_num, chart_content in chart_matches:
        lines = chart_content.strip().split('\n')
        title = ""
        description = ""
        
        for line in lines:
            line = line.strip()
            if not line:
                continue
            # 匹配"图X：标题"格式
            title_match = re.match(r'图\s*\d+[:：]\s*(.+)', line)
            if title_match:
                title = title_match.group(1).strip()
            elif title and not description:
                # 第一行非标题内容作为解析文本
                description = line
            elif description:
                # 后续内容追加到解析文本
                description += " " + line
        
        references.append({
            "number": int(chart_num),  # 图表编号（如1, 2, 3...）
            "title": title,            # 图表标题
            "description": description.strip()  # 图表解析文本
        })
    
    return references
```

## 五、图表与文本匹配流程

### 1. 匹配图表到引用（`_match_charts_to_references`）

**位置**：`backend/app/services/report_exporter.py` 第604-652行

**流程**：
```python
def _match_charts_to_references(self, charts: List[Dict], references: List[Dict]) -> List[Dict]:
    """根据文本引用匹配实际图表"""
    matched = []
    used_charts = set()
    
    # 1. 按引用顺序匹配
    for ref in references:
        best_match = None
        best_match_idx = -1
        
        # 2. 遍历所有图表，找到标题匹配的
        for idx, chart in enumerate(charts):
            if idx in used_charts:
                continue
            
            chart_title = chart.get("title", "")
            # 3. 尝试精确匹配或包含匹配
            if ref["title"] in chart_title or chart_title in ref["title"]:
                if best_match is None or len(chart_title) < len(best_match.get("title", "")):
                    best_match = chart
                    best_match_idx = idx
        
        # 4. 如果找到匹配，添加到结果中，并标记为已使用
        if best_match:
            matched.append({
                **best_match,
                "text_reference_number": ref["number"],  # 文本中的编号
                "text_reference_title": ref["title"]     # 文本中的标题
            })
            used_charts.add(best_match_idx)
    
    # 5. 添加未匹配的图表（按原始顺序）
    for idx, chart in enumerate(charts):
        if idx not in used_charts:
            matched.append(chart)
    
    # 6. 按引用编号排序
    matched.sort(key=lambda x: x.get("text_reference_number", 999))
    
    return matched
```

**匹配结果**：
- 匹配成功的图表会包含 `text_reference_number` 和 `text_reference_title`
- 未匹配的图表会追加到末尾

### 1. 构建图表HTML（`_build_charts_grid`）

**位置**：`backend/app/services/report_exporter.py` 第654-750行

```python
def _build_charts_grid(self, charts: List[Dict], section_content: str = None) -> str:
    # 1. 如果提供了章节内容，提取图表引用
    if section_content:
        references = self._extract_chart_references(section_content)
        if references:
            # 2. 构建描述映射：图表编号 -> 解析文本
            chart_descriptions = {}
            for ref in references:
                chart_descriptions[ref["number"]] = ref.get("description", "")
            
            # 3. 匹配图表到引用
            matched_charts = self._match_charts_to_references(charts, references)
    
    # 4. 遍历匹配后的图表，生成HTML
    for i, chart in enumerate(matched_charts):
        # 5. 获取图表编号（优先使用文本中的引用编号）
        if "text_reference_number" in chart:
            display_index = chart["text_reference_number"]
        else:
            display_index = i + 1
        
        # 6. 获取图表解析文本
        description = chart_descriptions.get(display_index, "")
        
        # 7. 生成HTML（包含图片和解析文本）
        item_html = f"""
        <div class="chart-item">
            <div class="chart-header">
                <h4>图 {display_index}：{title}</h4>
            </div>
            <div class="chart-image">
                <img src="{preview_image}" alt="{title}">
            </div>
            {f'<div class="chart-description">{description}</div>' if description else ''}
        </div>
        """
```

## 七、完整流程总结

### 流程图：

```
1. 专家执行阶段（WeatherExecutor/ComponentExecutor）
   ├─ 收集图表信息（从tool_results中提取visuals）
   ├─ 构建图表列表文本（"图1：标题 (类型)"）
   └─ 生成Prompt（包含图表信息和要求）

2. LLM生成阶段
   └─ LLM返回带预设标识的Markdown内容
      ├─ [WEATHER_SECTION_START]... [WEATHER_SECTION_END]
      └─ [CHART_X_START]... [CHART_X_END]（包含图表解析文本）

3. 报告导出阶段（ReportExporter）
   ├─ 解析预设标识（_parse_section_markers）
   │  └─ 提取章节内容，去除标识
   ├─ 提取图表引用（_extract_chart_references）
   │  └─ 从[CHART_X_START]... [CHART_X_END]中提取编号、标题、解析文本
   ├─ 匹配图表到引用（_match_charts_to_references）
   │  └─ 根据标题匹配，建立图表与文本的对应关系
   └─ 组装HTML（_build_charts_grid）
      └─ 生成包含图片和解析文本的HTML
```

### 关键点：

1. **预设标识的作用**：
   - `[WEATHER_SECTION_START]` / `[WEATHER_SECTION_END]`：标识章节边界
   - `[CHART_X_START]` / `[CHART_X_END]`：标识图表解析文本的位置

2. **图表编号的连续性**：
   - 专家阶段：从1开始编号（`chart_index = 1`）
   - 导出阶段：使用文本中的编号（`text_reference_number`）

3. **匹配策略**：
   - 优先精确匹配标题
   - 其次包含匹配（`ref["title"] in chart_title`）
   - 未匹配的图表追加到末尾

4. **解析文本提取**：
   - 从 `[CHART_X_START]` 和 `[CHART_X_END]` 之间提取
   - 第一行是"图X：标题"，后续行是解析文本
