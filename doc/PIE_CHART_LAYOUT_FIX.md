# 饼图布局优化

## 问题描述

用户反馈：行业VOCs排放贡献饼图的布局有问题：
1. 下面有部分在边框外面（标签溢出）
2. 图表图例和饼图本身都在左侧，视觉效果不好
3. 空间利用不合理

**之前的布局问题**:
```
┌────────────────────────────────────┐
│ 图例（左侧）                        │
│ ○ 机动车燃油等量                    │
│ ○ 专用博物馆加工                    │
│ ○ 塑料零件及其他塑料制品制造        │  ← 图例占据左侧
│                                    │
│        [饼图]                      │  ← 饼图在左侧
│                                    │
│  标签溢出边框外 ↓                  │
└────────────────────────────────────┘
```

## 解决方案

### 优化饼图布局为"左右分栏"模式

**改动**: `frontend/src/components/ChartsPanel.tsx` (Line 192-293)

#### 1. 改为环形图

**之前**:
```typescript
radius: '60%',  // 实心饼图，半径60%
```

**现在**:
```typescript
radius: ['40%', '60%'],  // 🔧 环形图，内半径40%，外半径60%
```

**优势**:
- 更紧凑，节省空间
- 视觉效果更现代
- 中间空白可以显示标题或统计信息（未来扩展）

#### 2. 饼图居中偏左

**之前**:
```typescript
// 未设置 center，默认 ['50%', '50%'] 完全居中
```

**现在**:
```typescript
center: ['40%', '50%'],  // 🔧 水平40%，垂直50%
```

**效果**:
- 饼图位于左侧 40% 位置
- 为右侧图例留出 60% 空间
- 垂直居中

#### 3. 图例移到右侧

**之前**:
```typescript
legend: {
  orient: 'vertical',  // 垂直
  left: 'left'         // 左对齐
}
```

**现在**:
```typescript
legend: {
  orient: 'vertical',  // 🔧 垂直布局
  right: '5%',         // 🔧 距右侧5%
  top: 'middle',       // 🔧 垂直居中
  align: 'left',       // 🔧 文字左对齐
  itemGap: 8,          // 🔧 图例项间距
  textStyle: {
    fontSize: 12
  },
  formatter: (name: string) => {
    // 🔧 限制图例文字长度，超过15字显示省略号
    return name.length > 15 ? name.slice(0, 15) + '...' : name
  }
}
```

**改动**:
- `left: 'left'` → `right: '5%'`: 从左侧移到右侧
- 添加 `top: 'middle'`: 垂直居中对齐
- 添加 `align: 'left'`: 图例标记与文字左对齐
- 添加 `itemGap: 8`: 图例项之间间距 8px
- 添加 `formatter`: 长文字自动截断

#### 4. 标签优化（避免溢出）

**之前**:
```typescript
// 使用默认标签配置，可能溢出
```

**现在**:
```typescript
label: {
  fontSize: 11,                    // 🔧 减小字号（默认12）
  formatter: '{b}: {d}%',          // 🔧 显示名称和百分比
  overflow: 'truncate',            // 🔧 超长标签截断
  width: 100                       // 🔧 标签最大宽度100px
},
labelLine: {
  length: 10,   // 🔧 缩短第一段引导线（默认15）
  length2: 5    // 🔧 缩短第二段引导线（默认10）
}
```

**优化点**:
- 减小字号：11px（更紧凑）
- 格式化标签：只显示名称和百分比（如 "机动车燃油等量: 75.72%"）
- 截断超长文本：避免标签溢出边框
- 缩短引导线：减少占用空间

#### 5. 添加标题支持

**新增**:
```typescript
title: title ? {
  text: title,
  left: 'center',  // 居中
  top: 10,
  textStyle: {
    fontSize: 15,
    fontWeight: 'bold',
    color: '#1f2328'
  }
} : undefined,
```

**效果**: 如果传入 title，显示在图表顶部居中

#### 6. 优化 Tooltip

**之前**:
```typescript
tooltip: { trigger: 'item' }
```

**现在**:
```typescript
tooltip: {
  trigger: 'item',
  formatter: '{b}: {c} ({d}%)'  // 🔧 显示：名称、数值、百分比
}
```

**显示格式**: "机动车燃油等量: 1234 (75.72%)"

## 技术细节

### 1. 环形图配置

**radius 参数**:
- 单值 `'60%'`: 实心饼图
- 双值 `['40%', '60%']`: 环形图
  - 第一个值: 内半径 (40%)
  - 第二个值: 外半径 (60%)

**视觉效果**:
```
   外半径 60%
    ↓───────
   ┌─────────┐
   │  ┌───┐  │
   │  │   │  │ ← 内半径 40%
   │  └───┘  │
   └─────────┘
```

### 2. 饼图定位

**center 参数**: `[水平位置, 垂直位置]`

```typescript
center: ['40%', '50%']
```

**坐标系统**:
```
  0%                    50%                   100%
   ├──────────────────────┼──────────────────────┤
   │                      │                      │
   │        40%           │                      │
   │         ↓            │                      │
   │        [饼图]        │      图例            │
   │                      │                      │
   └──────────────────────┴──────────────────────┘
```

### 3. 图例右对齐

**right 参数**: 距右边框的距离

```typescript
right: '5%'  // 距右边框 5%
```

**可选值**:
- `'left'`, `'center'`, `'right'`: 预定义位置
- `10`, `'10%'`: 像素或百分比
- 也可以用 `left` 参数从左边定位

**top: 'middle'**: 垂直居中
- 其他值: `'top'`, `'bottom'`, `10`, `'10%'`

### 4. 标签格式化

**formatter 参数**: 模板字符串

```typescript
formatter: '{b}: {d}%'
```

**模板变量**:
- `{a}`: 系列名称
- `{b}`: 数据项名称
- `{c}`: 数据值
- `{d}`: 百分比

**示例**:
- `'{b}: {d}%'` → "机动车燃油等量: 75.72%"
- `'{b}\n{c} ({d}%)'` → "机动车燃油等量\n1234 (75.72%)"

### 5. 标签溢出处理

**overflow 参数**:
- `'truncate'`: 截断（当前使用）
- `'break'`: 换行
- `'breakAll'`: 强制换行

**width 参数**: 标签最大宽度（像素）

```typescript
label: {
  overflow: 'truncate',
  width: 100  // 超过100px截断并显示省略号
}
```

### 6. 引导线长度

**labelLine 参数**:

```typescript
labelLine: {
  length: 10,   // 第一段（从饼图到转折点）
  length2: 5    // 第二段（从转折点到标签）
}
```

**示例**:
```
    饼图
     ●────── (length)
            └──── (length2) 标签文字
```

```typescript
  center: isMobile ? ['50%', '40%'] : ['40%', '50%'],  // 小屏居中，饼图上移

const legendConfig = {
  orient: isMobile ? 'horizontal' : 'vertical',  // 小屏水平布局

  [isMobile ? 'left' : 'top']: isMobile ? 'center' : 'middle'
}
```
