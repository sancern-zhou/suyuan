# 大气污染溯源分析系统 - 前端重构启动指南

## 🎯 快速启动

### 方式一：使用启动脚本（推荐）

**Windows用户**:
```bash
# 双击运行
D:\溯源\frontend\start.bat
```

**Linux/Mac用户**:
```bash
# 终端执行
cd D:\溯源/frontend
chmod +x start.sh
./start.sh
```

### 方式二：手动启动

```bash
# 1. 进入前端目录
cd D:\溯源\frontend

# 2. 安装依赖（首次运行）
npm install

# 3. 启动开发服务器
npm run dev
```

### 访问地址

启动后，在浏览器中访问：
```
http://localhost:5174
```

## 🔧 常见问题

### Q1: 端口被占用
```bash
# 解决方案：修改vite.config.js中的端口
server: {
  port: 5175  # 改为其他端口
}
```

### Q2: 依赖安装失败
```bash
# 解决方案：清理缓存重新安装
npm cache clean --force
rm -rf node_modules
npm install
```

### Q3: 图表不显示
确保ECharts已正确安装：
```bash
npm list echarts
```

### Q4: 样式异常
检查SCSS文件是否正确导入：
```javascript
// src/main.js 中应包含
import './styles/main.scss'
```

## 📞 技术支持

如遇到问题，请检查：
1. Node.js版本 >= 16
2. npm版本 >= 8
3. 端口5174未被占用
4. 防火墙未阻止访问
